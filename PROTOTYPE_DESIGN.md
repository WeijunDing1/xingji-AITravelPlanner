# 行迹 TripCraft — 原型设计说明

## 设计原则

- **移动端优先**：按 375x812（iPhone X）设计，Web 端居中卡片展示
- **极简操作**：首页到结果最多 2 步
- **信息分层**：概览 → 详情逐步展开，避免信息过载
- **地图驱动**：地图不只是装饰，是核心交互载体

## 视觉风格指南

### 配色方案
| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | #2563EB → #7C3AED | 蓝紫渐变，现代感 |
| 辅助色 | #10B981 | 绿色，用于成功/金额 |
| 背景 | #F8FAFC | 浅灰白底 |
| 卡片 | #FFFFFF | 纯白卡片，带轻微阴影 |
| 文字主色 | #1E293B | 深灰黑 |
| 文字辅助 | #64748B | 中灰 |

### 字体
- 标题：SF Pro Display / PingFang SC Medium
- 正文：SF Pro Text / PingFang SC Regular
- 数字：DIN Alternate（预算数字更好看）

### 组件风格
- 卡片：圆角 12px，阴影 `0 2px 8px rgba(0,0,0,0.06)`
- 按钮：圆角 24px（胶囊型），高度 48px
- 标签：圆角 16px，高度 32px，选中填充主色
- 间距：统一使用 8px 倍数（8/12/16/24/32）

### 图标
- 使用 Lucide Icons（线性风格）
- POI 类型图标：景点🏛 / 餐厅🍜 / 住宿🏨 / 交通🚗
- 可用 emoji 代替（MVP阶段降低开发成本）

---

## 响应式策略

| 屏幕宽度 | 布局 |
|----------|------|
| < 480px | 全宽移动端布局（主设计） |
| 480-768px | 居中 max-width: 480px |
| > 768px | 居中卡片式，两侧留白，max-width: 420px |

---

## 动效建议

| 场景 | 动效 |
|------|------|
| 页面切换 | 左右滑动过渡（300ms ease） |
| 标签选中 | 缩放弹性效果（scale 0.95 → 1） |
| 卡片展开 | 高度渐变 + fade in（200ms） |
| Loading步骤 | 逐个淡入 + 打勾动画 |
| 地图POI出现 | 从上方落下弹跳效果 |
| Tab切换 | 下划线滑动跟随 |

---

## 关键交互细节

1. **文本框**：支持 Shift+Enter 换行，Enter 不提交（移动端无此问题）
2. **图片上传**：上传后显示缩略图 + 删除按钮，支持预览大图
3. **Day 切换**：横向滑动切换，当前日期居中高亮
4. **地图交互**：点击 POI 标注弹出简要卡片，再点击跳转详情
5. **分享**：生成带二维码的图片海报 / 复制链接
6. **重新生成**：保留当前输入，可微调后重新生成
